 🏦 Executive Banking Analytics Dashboard
📌 Project Overview

The Executive Banking Analytics Dashboard is an interactive Microsoft Excel dashboard designed to analyze loan portfolio performance, customer demographics, lending trends, and risk indicators. The dashboard transforms raw banking data into actionable insights using Pivot Tables, Pivot Charts, KPI Cards, Slicers, and advanced Excel analytics techniques.

This project enables users to monitor key banking metrics, evaluate loan performance, identify risk patterns, and gain a comprehensive view of lending operations through an intuitive executive-level reporting interface.



🎯 Objectives

- Analyze loan application and funding trends.
- Monitor loan portfolio performance.
- Evaluate customer segmentation based on income and home ownership.
- Track loan repayment and default patterns.
- Identify revenue-generating regions and customer groups.
- Provide interactive filtering and drill-down capabilities.
